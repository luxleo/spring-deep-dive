# Thread Local

클래스의 변수를 참조할때 동시성 문제를 해결하기 위해 사용하는 자료구조이다.

Thread local 은 스레드 별로 값을 저장하는 자료구조이다. 각 스레드는 고유한 값을 가지며, 다른 스레드는 해당 스레드의 값을 볼 수 없다.

다만 주의할 점은 ThreadLocal.remove() 를 호출하지 않으면 WAS 가 사용하는 스레드풀의 스레드가 재사용될 때, 이전 스레드의 값을 남겨두게 된다. 
이는 예상치 못한 동작을 일으킬 수 있으므로, 사용 후 반드시 ThreadLocal.remove() 를 호출하여 스레드 로컬 변수를 제거해야 한다.
